### 1、为什么选择JSON库？

​	因为它足够简单，除了基本知识外不需要大量技术背景知识。JSON有自己的标准，可以按照标准逐步实现。JSON也是在实际中大量应用的格式，所以存在大量的开源库。

### 2、本项目拟定支持的特性

从零手写一个JSON库，其特性如下：

- 符合标准的JSON解释器和生成器；
- 手写的递归下降解析器（recursive descent parser）
- 使用C++11
- 支持跨平台（Windows/Linux）
- 仅支持UTF-8 JSON文本
- 仅支持以 `double` 存储 JSON number 类型
